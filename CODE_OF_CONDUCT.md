# Code of Conduct

We follow the Contributor Covenant (v2.1). By participating, you are expected to uphold this code.

- Be respectful and inclusive.
- Assume good intent; give constructive feedback.
- No harassment or discrimination of any kind.

Enforcement: Report unacceptable behavior to the maintainers at security@plugged.in.

For the full text, see https://www.contributor-covenant.org/version/2/1/code_of_conduct.html
